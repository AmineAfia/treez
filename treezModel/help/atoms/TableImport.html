TableImport

Purpose

The purpose of the atom is to import external data as a result table. The development of this atom is not yet finished.

Class name

org.treez.model.atom.executable.TableImport

Construction

You can create a new atom:
  • with the context menu of an existing atom or
  • by calling the corresponding factory method of an atom: 
    TableImport tableImport = executable.createTableImport("tableImport");
