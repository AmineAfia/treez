Models

Purpose

The atom is the parent atom for all underlying model atoms. Creating the is typically the first step in a simulation work flow. Each individual model atom (the children of the atom) implements the interface org.treez.model.interfaces.Model.

Class name

org.treez.model.atom.Models

Construction

A new atom is created either
  • from the context menu of an existing atom or
  • by calling the corresponding factory method of the atom: 
    Models models = root.createModels("models");
