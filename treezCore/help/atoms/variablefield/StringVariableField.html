StringVariableField

Purpose

The purpose of the is to edit a String variable. It can be used by a .

Class name

org.treez.core.atom.variablefield.StringVariableField

Construction

A new is created by:
  • using the context menu of a atom or
  • by calling the corresponding factory method of the atom: 
    StringVariableField myString = genericModel.createStringVariableField("myString");

User interaction

Each VariableField provides two controls:
  • a so called AttributeAtomControl that is used to edit a variable (e.g. in the control of a ).
  • a control that allows to edit the properties of the VariableField itself (e.g. the label and the tool tip that are used in the AttributeAtomControl.).

The check box of the AttributeAtomControl enables/disables the VariableField. The context menu in the also allows to enable/disable the VariableField and to delete it.

Attributes

  • Label: the label that is used in the AttributeAtomControl.
  • DefaultValueString: the default value of the variable. This value is used if the VariableField is reset.
  • Tooltip: the tool tip that is used in the AttributeAtomControl.
